# Code of Conduct

## Expected behavior

- Be respectful and constructive in issues, pull requests, and discussions.
- Keep contributions focused on the stated scope of this project.
- Assume good faith when asking about unexpected behavior or limitations.
- Follow the contribution guidelines in [CONTRIBUTING.md](.github/CONTRIBUTING.md).

## Unacceptable behavior

- Harassment, personal attacks, or threats of any kind.
- Posting content that is intentionally harmful, discriminatory, or unlawful.
- Intentionally disrupting or derailing project discussions.
- Submitting content that contains real credentials, personal data, or sensitive material.

## Reporting

If you witness or experience unacceptable behavior, contact the project maintainer privately rather than posting publicly. Security vulnerabilities should follow [SECURITY.md](SECURITY.md) instead of this process.

## Scope

This code of conduct applies to all project spaces: GitHub Issues, Pull Requests, Discussions, and any other channel where this project is represented.
